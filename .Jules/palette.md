## 2025-05-15 - Out of Service State for Amenities
**Learning:** Amenities in a physical location (like changing tables) can be present but unusable. Representing this 'out_of_service' state with a distinct visual (orange tint) and a clear text suffix improves user trust and prevents wasted trips.
**Action:** Always consider the "exists but broken" state for physical amenities and provide clear visual and textual feedback.
